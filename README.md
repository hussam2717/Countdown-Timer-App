# Countdown-Timer-App
A simple and interactive Countdown Timer built using modern web technologies. This app allows users to set a future date and time and displays the remaining time in days, hours, minutes, and seconds.

🚀 Features
⏱️ Live countdown timer
📅 Set custom date and time
🔄 Real-time updates every second
🎯 Clean and responsive UI
⚡ Fast and lightweight

🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6+)
