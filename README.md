# Countdown-Timer-App
A simple and interactive Countdown Timer built using modern web technologies. This app allows users to set a future date and time and displays the remaining time in days, hours, minutes, and seconds.

live demo -  https://hussam2717.github.io/Countdown-Timer-App/

🚀 Features
⏱️ Live countdown timer
📅 Set custom date and time
🔄 Real-time updates every second
🎯 Clean and responsive UI
⚡ Fast and lightweight

🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6+)

💡 How It Works
User selects a future date & time
App calculates the difference between current time and target time
Updates countdown every second using JavaScript setInterval

📌 Future Improvements
🔔 Notification when time ends
🎨 Theme customization (dark/light mode)
📱 Mobile optimization
🧠 Save countdown using local storage
